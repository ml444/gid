# gid
Global ID Component Based on Snowflake Algorithm.

## ID的位分配模式
1. 经典的雪花模式
    ```
    【经典雪花模式】：
    | 版本 | 豪秒级时间差 | 机器ID | 序列号 |
    | 63  | 62-22      | 21-12 | 11-0   |
    ```

2. 最大峰值模式
    ```
    【最大峰值】：
    | 版本 | 生成方式 | 秒级时间差 | 机器ID | 序列号 |
    | 63  | 62-61   | 60-30    | 29-20 | 19-0   |
    ```

3. 最小粒度模式
    ```
    【最小粒度】：
    | 版本 | 生成方式 | 豪秒级时间差 | 机器ID | 序列号 |
    | 63  | 62-61   | 60-20      | 19-10 | 9-0   |
    ```
